Ejercicio 1 – SPA
- Simula una página única.
- Cambia el contenido sin recargar usando history.pushState().

Ejercicio 2 – Panel de Diagnóstico
- Muestra tamaño ventana, pantalla, posición y estado de conexión.
- Se actualiza automáticamente con resize, online/offline y movimiento.
- Archivos: index.html, diagnostico.js

Ejercicio 3 – Scroll y Volver Arriba
- Barra de progreso de scroll y botón para volver al inicio.
- Animación suave de aparición del botón.
- Archivos: index.html, scroll.js, style.css

Ejercicio 4 – Pop-ups
- Abre ventana emergente automática o manual.
- Se puede cerrar desde la página principal o el pop-up.
- El pop-up cambia texto en la ventana principal.
- Archivos: index.html, popup.html, popup-main.js
