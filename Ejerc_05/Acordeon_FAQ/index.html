Preguntas Frecuentes (FAQ)

  • ¿Qué es JavaScript?

    JavaScript es un lenguaje de programación que permite crear interactividad en las páginas web.

  • ¿Qué es una función?

    Una función es un bloque de código que realiza una tarea específica y puede ser reutilizado.

  • ¿Qué es el DOM?

    El DOM es la estructura del documento HTML que el navegador interpreta y permite manipular con JavaScript.
